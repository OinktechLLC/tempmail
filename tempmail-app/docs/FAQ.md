# FAQ — Часто задаваемые вопросы

## Общие вопросы

### ❓ Что такое TempMail?

**TempMail** — это сервис временной электронной почты. Вы получаете уникальный адрес, который действует 90 дней пока вы активны. Идеально для регистраций на сайтах без раскрытия личного email.

### ❓ Как работает система 90 дней?

- При первом посещении создаётся временный адрес
- Запускается таймер на 90 дней
- **Любое действие** (клик, движение мыши, открытие письма) продлевает срок на 90 дней
- Если не заходить 90 дней — адрес автоматически меняется на новый
- Если активны — срок продлевается бесконечно

### ❓ Это бесплатно?

Да! Сервис полностью бесплатный, без скрытых платежей и регистрации.

### ❓ Нужно ли регистрироваться?

Нет, регистрация не требуется. Адрес генерируется автоматически при первом посещении.

## Использование

### ❓ Как получить свой временный адрес?

Просто откройте сайт — адрес сгенерируется автоматически при принятии условий использования.

### ❓ Как скопировать адрес?

Нажмите кнопку **«📋 Копировать»** в правой панели. Адрес скопируется в буфер обмена.

### ❓ Как проверить новые письма?

Письма обновляются автоматически каждые 30 секунд. Также можно нажать кнопку **«⟳ Обновить»**.

### ❓ Почему письма не приходят?

Возможные причины:
- Письмо ещё не доставлено (подождите 1-2 минуты)
- Отправитель использует блокировку временных почт
- Проблема на стороне API 1secmail
- Письмо попало в спам (проверьте папку «Спам»)

### ❓ Как продлить срок действия адреса?

Нажмите кнопку **«↻ Продлить на 90 дней»** в правой панели. Любое действие также автоматически продлевает срок.

### ❓ Как создать новый адрес?

Нажмите кнопку **«🔄 Создать новый адрес»**. Старый адрес будет потерян вместе с письмами.

## Спам и фильтрация

### ❓ Что такое папка «Спам»?

Автоматический фильтр определяет подозрительные письма по ключевым словам:
- spam, lottery, winner, casino, crypto, investment
- привет, заработок и другие

### ❓ Можно ли отключить фильтр спама?

В текущей версии фильтр работает автоматически. Письма не удаляются, а перемещаются в папку «Спам».

### ❓ Как отметить письмо как важное?

Нажмите на звёздочку ⭐ рядом с письмом — оно попадёт в папку «Избранное».

## Безопасность и приватность

### ❓ Безопасно ли использовать TempMail?

Для временных задач — да. Но помните:
- ⚠️ Не используйте для важных аккаунтов (банки, госуслуги)
- ⚠️ Не передавайте конфиденциальную информацию
- ⚠️ Письма хранятся у стороннего провайдера (1secmail)

### ❓ Где хранятся мои данные?

Все данные хранятся **локально в вашем браузере** (localStorage):
- Временный адрес
- Дата создания и активности
- Список писем (метаданные)

Мы не передаём данные третьим лицам (кроме запросов к API 1secmail).

### ❓ Кто может видеть мои письма?

- Вы (на своём устройстве)
- Провайдер API 1secmail (на своих серверах)
- Любой, кто имеет доступ к вашему браузеру

### ❓ Что происходит при очистке кеша браузера?

Все данные будут потеряны:
- Временный адрес удалится
- При следующем посещении создастся новый
- Старые письма будут недоступны

### ❓ Можно ли восстановить старый адрес?

Нет. После очистки данных или смены адреса восстановление невозможно.

## Технические вопросы

### ❓ Какой API используется?

Сервис использует бесплатный API [1secmail](https://www.1secmail.com/):
- Не требует регистрации
- Без ключей API
- Поддерживает несколько доменов

### ❓ Какие домены доступны?

Доступны домены 1secmail:
- 1secmail.com
- 1secmail.org
- 1secmail.net

Домен выбирается случайно при генерации адреса.

### ❓ Работает ли на мобильных устройствах?

Да! Интерфейс адаптирован для:
- Смартфонов (iOS, Android)
- Планшетов
- Десктопов

### ❓ Какие браузеры поддерживаются?

Все современные браузеры:
- Chrome / Chromium
- Firefox
- Safari
- Edge
- Opera

### ❓ Можно ли использовать свой домен?

В текущей версии — нет. Используются только домены 1secmail.

### ❓ Есть ли API для разработчиков?

Вы можете напрямую использовать API 1secmail. Документация: https://www.1secmail.com/api/

## Ограничения

### ❓ Сколько писем можно получить?

Ограничений нет, но помните о лимитах API 1secmail.

### ❓ Как долго хранятся письма?

Срок хранения определяется политикой 1secmail. Обычно несколько дней.

### ❓ Можно ли отправлять письма?

Нет, сервис только для **получения** писем.

### ❓ Можно ли ответить на письмо?

Нет, функция отправки не доступна.

### ❓ Работает ли с вложениями?

API 1secmail поддерживает вложения, но в текущей версии они не отображаются.

## Проблемы и решения

### ❓ Чёрный экран при загрузке

**Решение:**
1. Очистите кеш браузера
2. Отключите блокировщики рекламы
3. Проверьте консоль разработчика (F12) на ошибки
4. Попробуйте другой браузер

### ❓ Письма не обновляются

**Решение:**
1. Нажмите кнопку «Обновить»
2. Проверьте подключение к интернету
3. Подождите 1-2 минуты
4. Перезагрузите страницу

### ❓ Таймер показывает «Истекло»

Это значит, что прошло 90 дней без активности. Адрес автоматически заменён на новый.

### ❓ Модальное окно не исчезает

Убедитесь, что отметили обе галочки:
- ☑️ Принять Условия использования
- ☑️ Принять Политику конфиденциальности
