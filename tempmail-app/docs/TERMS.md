# Условия использования сервиса TempMail

**Последнее обновление:** 28 мая 2026 г.

## 1. Общие положения

1.1. Настоящие Условия использования (далее — «Условия») регулируют использование сервиса временной электронной почты TempMail (далее — «Сервис»).

1.2. Используя Сервис, вы подтверждаете, что прочитали, поняли и принимаете настоящие Условия в полном объёме.

1.3. Если вы не согласны с Условиями, пожалуйста, не используйте Сервис.

1.4. Администрация Сервиса оставляет за собой право изменять Условия в любое время без предварительного уведомления.

## 2. Описание Сервиса

2.1. TempMail предоставляет временные адреса электронной почты для получения писем без регистрации.

2.2. **Срок действия адреса:**
- Временный адрес действует 90 дней с момента последней активности пользователя
- Любое действие пользователя (просмотр писем, копирование адреса, продление) обновляет срок действия на 90 дней
- При отсутствии активности в течение 90 дней адрес автоматически заменяется на новый

2.3. Сервис использует сторонний API (1secmail) для получения писем.

## 3. Правила использования

3.1. **Разрешённое использование:**
- Получение писем для одноразовых регистраций
- Тестирование функциональности email-рассылок
- Защита личного email от спама
- Временная переписка низкой важности

3.2. **Запрещённое использование:**
- Рассылка спама или нежелательных сообщений
- Мошенническая деятельность
- Нарушение авторских прав
- Распространение вредоносного ПО
- Обход блокировок и ограничений других сервисов
- Использование для важных аккаунтов (банки, госуслуги, медицинские услуги)
- Хранение конфиденциальной информации
- Нарушение законодательства вашей страны
