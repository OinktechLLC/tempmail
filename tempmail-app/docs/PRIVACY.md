# Политика конфиденциальности TempMail

**Последнее обновление:** 28 мая 2026 г.

## 1. Введение

1.1. Настоящая Политика конфиденциальности (далее — «Политика») описывает, как сервис TempMail (далее — «Сервис», «мы», «нас») собирает, использует, хранит и защищает информацию пользователей.

1.2. Мы стремимся минимизировать сбор данных и защищать вашу приватность.

## 2. Какие данные мы собираем

### 2.1. Данные, хранимые локально (в вашем браузере)

Мы используем localStorage вашего браузера для хранения:

- **Временный email адрес** — сгенерированный адрес электронной почты
- **Логин и домен** — компоненты адреса для работы с API
- **Дата создания** — время первого создания адреса
- **Дата последней активности** — время последнего действия пользователя
- **Статус принятия условий** — факт согласия с Условиями и Политикой

### 2.2. Данные, получаемые от сторонних сервисов

- **Письма** — содержимое писем, полученных через API 1secmail
- **Метаданные писем** — отправитель, тема, дата получения

### 2.3. Данные, которые мы НЕ собираем

Мы НЕ собираем и НЕ храним:

- Личные данные (имя, фамилия, телефон)
- IP-адреса
- Геолокацию
- Историю посещений
- Данные об устройстве
- Пароли или учётные данные
- Платёжную информацию

## 3. Как мы используем данные

### 3.1. Основные цели использования

- **Функционирование Сервиса** — отображение вашего временного адреса и писем
- **Отслеживание активности** — определение срока действия адреса (90 дней)
- **Автоматическая смена адреса** — генерация нового при неактивности
- **Улучшение пользовательского опыта** — сохранение сессии между посещениями

### 3.2. Автоматическое отслеживание активности

Сервис автоматически отслеживает следующие действия для продления срока действия адреса:

- Движение мыши
- Клики
- Нажатия клавиш
- Прокрутка страницы
- Касания (на мобильных устройствах)
- Открытие писем
- Копирование адреса
- Продление срока действия

## 4. Хранение данных

### 4.1. Локальное хранение

Все данные хранятся исключительно в localStorage вашего браузера:

- Данные не передаются на наши серверы
- Данные доступны только вам
- Данные удаляются при очистке кеша браузера

### 4.2. Сторонние сервисы

- **1secmail API** — письма временно хранятся на серверах 1secmail
- Срок хранения писем у 1secmail определяется их политикой
- Мы не контролируем хранение данных у третьих лиц

### 4.3. Срок хранения

- **Данные в браузере** — хранятся пока вы не очистите localStorage
- **Адрес действует** — 90 дней с последней активности
- **Письма у провайдера** — определяются политикой 1secmail

## 5. Передача данных третьим лицам

### 5.1. Когда мы передаём данные

Мы передаём данные только в следующих случаях:

- **API 1secmail** — запросы для получения ваших писем
- **Требование закона** — по официальному запросу правоохранительных органов

### 5.2. Когда мы НЕ передаём данные

Мы НЕ передаём данные:

- Рекламным сетям
- Маркетинговым компаниям
- Страховым компаниям
- Работодателям
- Другим пользователям

## 6. Безопасность данных

### 6.1. Наши меры безопасности

- Минимизация собираемых данных
- Локальное хранение без передачи на сервер
- Использование HTTPS для всех запросов
- Отсутствие отслеживания и аналитики

### 6.2. Ваши меры безопасности

Для защиты данных рекомендуем:

- Не использовать для важной переписки
- Не передавать конфиденциальную информацию
- Регулярно очищать данные браузера
- Использовать приватный режим при необходимости

### 6.3. Ограничения безопасности

Помните о рисках:

- Временная почта не предназначена для секретной информации
- Письма могут быть доступны через API провайдера
- Данные в браузере доступны другим пользователям устройства

## 7. Права пользователей

Вы имеете право:

### 7.1. Право на доступ

- Просматривать все хранящиеся данные в браузере
- Проверять полученные письма

### 7.2. Право на удаление

- Удалить данные через очистку localStorage
- Сгенерировать новый адрес (старые данные потеряются)
- Очистить кеш браузера

### 7.3. Право на перенос

- Скопировать ваш временный адрес
- Сохранить важные письма отдельно

### 7.4. Право на отзыв согласия

- Прекратить использование Сервиса в любое время
- Очистить все данные через настройки браузера

## 8. Файлы cookie и аналогичные технологии

### 8.1. Используемые технологии

- **localStorage** — для хранения основных данных
- **sessionStorage** — не используется
- **Cookie** — не используются

### 8.2. Управление хранением

Вы можете управлять данными через:

- Настройки браузера
- Расширения для очистки данных
- Приватный режим просмотра

## 9. Изменения в Политике

### 9.1. Обновление Политики

- Мы можем обновлять Политику время от времени
- Новая версия публикуется на этой странице
- Дата последнего обновления указывается в начале

### 9.2. Уведомление об изменениях

- При значительных изменениях потребуется повторное согласие
- Продолжение использования означает принятие новой Политики

## 10. Контакты

### 10.1. Вопросы по конфиденциальности

По вопросам, связанным с Политикой конфиденциальности:

- Откройте Issue на GitHub
- Укажите тему «Privacy Question»

### 10.2. Жалобы

Если вы считаете, что мы нарушаем вашу приватность:

- Свяжитесь через GitHub Issues
- Опишите проблему подробно
- Мы ответим в разумные сроки

## 11. Применимое законодательство

### 11.1. Соответствие законам

Политика составлена с учётом:

- Федерального закона РФ № 152-ФЗ «О персональных данных»
- Общих принципов защиты приватности

### 11.2. Юрисдикция

Политика регулируется законодательством Российской Федерации.
