# TempMail - Временная почта на 90 дней

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![React](https://img.shields.io/badge/React-18.2-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-5.0-purple.svg)](https://vitejs.dev/)

**TempMail** — это современный сервис временной электронной почты с уникальной системой активности. Ваш адрес действует **90 дней**, пока вы активны. Любое действие продлевает срок ещё на 90 дней!

## 🌟 Особенности

- **📧 Временная почта на 90 дней** — адрес действует пока вы активны
- **🔄 Автоматическое продление** — любое действие обновляет таймер на 90 дней
- **⚠️ Фильтр спама** — подозрительные письма автоматически в спаме
- **⭐ Избранное** — отмечайте важные письма звёздочкой
- **🔒 Конфиденциальность** — данные хранятся только в вашем браузере
- **🎨 Дизайн как Gmail** — привычный и удобный интерфейс
- **📱 Адаптивность** — работает на всех устройствах
- **🆓 Бесплатно** — без регистрации и скрытых платежей

## 📖 Как это работает

### Система активности

1. **При первом посещении** создаётся уникальный временный адрес
2. **Таймер 90 дней** запускается с момента создания
3. **Любое действие** (клик, движение мыши, открытие письма) продлевает срок на 90 дней
4. **Если не активны 90 дней** — адрес автоматически меняется на новый
5. **Если активны** — срок продлевается бесконечно

### API

Сервис использует бесплатный API [1secmail](https://www.1secmail.com/) для получения писем:
- Не требует регистрации
- Без ключей API
- Работает стабильно

## 🔐 Безопасность и приватность

### Что мы храним
- Временный адрес (в localStorage браузера)
- Дата создания и последней активности
- Полученные письма

### Что мы НЕ храним
- Личные данные пользователя
- IP адреса
- История посещений
- Пароли

### Важные предупреждения
⚠️ **Не используйте для:**
- Банковских аккаунтов
- Государственных услуг
- Важной переписки
- Конфиденциальной информации
